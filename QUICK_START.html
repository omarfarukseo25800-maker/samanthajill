🔍 How to Search Properties

Quick Search

  1. Type in the search box at the top
  2. Search by: address, neighborhood, or zip code
  3. Results update instantly as you type

Advanced Filters (Left Sidebar)

Filter What It Does How to Use
Price Range Filter by min/max price Use sliders to set range
Bedrooms Minimum number of beds Check desired number(s)
Status Active or all listings Select from dropdown

Apply Filters

  1. Set your desired filters
  2. Click "Apply Filters" button
  3. Results update to match your criteria

🏡 View Property Details

Each property card shows:

  • 📍 Full address
  • 💰 Listing price
  • 🛏️ Number of bedrooms
  • 🚿 Number of bathrooms
  • 📏 Square footage
  • 🖼️ Property image

Get More Information

  1. Click on any property card
  2. Full details open in a popup window
  3. View phone number to contact landlord/agent
  4. Click "View Original" to see listing on original website

Quick Actions on Each Card

  • 👁️ View Details - See full information
  • 📞 Call - Dial phone number directly
  • 🔗 Visit Listing - Go to original website

📋 Change View Preference

Click the view toggle in the top right:

  • Grid View (📊 icon) - See properties as cards (default)
  • List View (📝 icon) - See properties as a vertical list

Choose whichever view you prefer for easier browsing!

📥 Download Data to Spreadsheet

Export as CSV

  1. Click "Export CSV" button in left sidebar
  2. File downloads to your computer
  3. Open with Excel, Google Sheets, or Numbers

What's Included in Export

  • Full address
  • Phone number
  • Price
  • Bedrooms & bathrooms
  • Square footage
  • Property type
  • Link to original listing
  • Date scraped
💡 Tip: Export weekly to track new listings and price changes

⚠️ Troubleshooting

Dashboard Won't Load

  • Check your internet connection
  • Try refreshing the page (F5)
  • Clear browser cache (Ctrl+Shift+Delete)
  • Try a different browser

Properties Not Showing

  • Click "Refresh Data" button
  • Wait 5-10 seconds for loading
  • Check that filters aren't too restrictive
  • Try clearing all filters

Can't Call a Property

  • Phone number may not be available
  • Try clicking "View Original" to see on main website
  • Some listings may have contact form instead

Export Not Working

  • Check pop-up settings in browser
  • Try a different browser
  • Ensure you have storage space on computer
  • File downloads to your "Downloads" folder
